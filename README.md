# API Test Automation with Newman & GitHub Actions

This project demonstrates automated API testing using **Postman collections**, executed via **Newman**, and integrated into a CI/CD pipeline using **GitHub Actions**.

---

## Project Overview

- API tests created in Postman
- Tests exported as collections
- Executed using Newman (CLI)
- Automated with GitHub Actions pipeline
- HTML reports generated for test results

---
## Tools & Technologies

- Postman
- Newman
- Node.js
- GitHub Actions

---
